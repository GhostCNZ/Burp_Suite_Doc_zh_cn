**注意:** 与任何其他的安全测试软件一样，Burp Suite包含的功能有可能对目标系统造成损坏。在对目标进行安全漏洞方面的的测试及交互过程可能导致目标出现某些问题。你应当在使用Burp的过程注意这些问题，建议在使用前阅读相关文档，了解测试中存在的风险，并且在测试前备份目标系统。不要对未进行授权的目标系统使用 Burp 进行安全测试。

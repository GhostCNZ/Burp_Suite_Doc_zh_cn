# 贡献力量
---------
如果你想对本书做出一些贡献的话

- 可以帮忙校对，找bug错别字等等
- 可以提出专业方面的修改建议
- 可以把一些不尽人意的语句翻译的更好更有趣

对于以上各类建议，请以issue的形式发送，我看到之后会尽快处理

# 如何参与
---------
将本书fork/pull到本地  
新建一个分支，然后联系我认领要翻译的章节  
使用MarkDown编辑器，md语法格式进行文档翻译及排版工作  
完成之后Pull Request该分支  
如没问题的话，我会合并到主分支  
如果不太明白md的排版，可以把翻译好的内容发送给我，我代为排版并提交  
如有其它问题，欢迎发送issue，谢谢~
